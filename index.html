موضوعات درسی

برای مشاهده مطالب هر درس، روی دکمه مربوطه کلیک کنید. تمامی مطالب به صورت طبقه‌بندی شده و با مثال‌های عملی ارائه شده‌اند.

طراحی سایت

یادگیری اصول طراحی وب‌سایت با HTML، CSS و JavaScript. ایجاد صفحات وب تعاملی و ریسپانسیو برای دستگاه‌های مختلف.

سرفصل‌های اصلی:

  • آشنایی با HTML و ساختار صفحات وب
  • استایل‌دهی با CSS و Flexbox
  • طراحی ریسپانسیو و واکنش‌گرا
  • برنامه‌نویسی مقدماتی با JavaScript
  • ایجاد تعامل با کاربر
  • آشنایی با فریمورک‌های CSS

پایگاه داده

آموزش مفاهیم پایگاه داده، طراحی و پیاده‌سازی جداول، نوشتن کوئری‌های SQL و مدیریت داده‌ها در سیستم‌های مختلف.

سرفصل‌های اصلی:

  • مفاهیم پایه و معماری پایگاه داده
  • طراحی مدل ER و نرمال‌سازی
  • دستورات اصلی SQL (SELECT, INSERT, UPDATE, DELETE)
  • انواع JOIN و توابع aggregate
  • مدیریت کاربران و دسترسی‌ها
  • آشنایی با MySQL و SQLite
