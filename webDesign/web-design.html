سرفصل‌های دوره طراحی سایت

در این دوره با تکنولوژی‌های اصلی طراحی وب آشنا می‌شوید و توانایی ساخت وب‌سایت‌های مدرن و تعاملی را کسب خواهید کرد.

HTML

زبان نشانه‌گذاری ابرمتن، اسکلت و ساختار اصلی صفحات وب را تشکیل می‌دهد.

مباحث اصلی:

  • آشنایی با تگ‌های اصلی HTML
  • فرم‌ها و کنترل‌های ورودی
  • جداول و لیست‌ها
  • ساختار معنایی HTML5
  • متریال و رسانه‌ها

CSS

زبان شیوه‌نامه‌ای برای توصیف presentation اسناد نوشته شده با HTML.

مباحث اصلی:

  • انتخاب‌گرها و ویژگی‌های پایه
  • Flexbox و Grid Layout
  • انیمیشن و transition
  • طراحی ریسپانسیو
  • فریمورک‌های CSS مانند Bootstrap

JavaScript

زبان برنامه‌نویسی برای ایجاد تعامل و پویایی در صفحات وب.

مباحث اصلی:

  • متغیرها، انواع داده و عملگرها
  • توابع و حوزه‌بندی
  • مدل شیء文档 (DOM)
  • رویدادها و مدیریت آن‌ها
  • AJAX و ارتباط با سرور
  • ES6 و ویژگی‌های مدرن

Node.js

محیط اجرای جاوااسکریپت سمت سرور برای ساخت برنامه‌های تحت وب مقیاس‌پذیر.

مباحث اصلی:

  • آشنایی با Node.js و NPM
  • ماژول‌ها و مدیریت وابستگی‌ها
  • ایجاد سرور HTTP
  • کار با فایل‌سیستم
  • Express.js و ساخت API
  • اتصال به پایگاه داده

SQL

زبان استاندارد برای مدیریت و کار با پایگاه‌های داده رابطه‌ای.

مباحث اصلی:

  • مفاهیم پایگاه داده رابطه‌ای
  • دستورات SELECT، INSERT، UPDATE، DELETE
  • فیلتر کردن و مرتب‌سازی داده‌ها
  • اتصالات (JOIN) و توابع aggregate
  • نرمال‌سازی و طراحی پایگاه داده
