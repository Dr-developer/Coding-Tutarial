درس اول: مبانی پایه JavaScript

آشنایی با مفاهیم اساسی و ساختار زبان

مبانی پایه زبان برنامه‌نویسی JavaScript

در این درس با مفاهیم اساسی JavaScript آشنا می‌شویم و اولین برنامه‌های خود را می‌نویسیم.

۱. جاوااسکریپت چیست؟

جاوااسکریپت زبانی است که رفتار (Behavior) صفحات وب را کنترل می‌کند.

HTML اسکلت است، CSS لباس، و جاوااسکریپت حرکات و هوش صفحه.

نکته مهم:

این زبان:

  • در مرورگر اجرا می‌شود (Chrome, Firefox, Edge و …)
  • روی Node.js خارج از مرورگر هم اجرا می‌شود
  • زبان پویا (Dynamic) و تفسیری (Interpreted) است
  • با امکانات کم شروع می‌شود ولی تا ساخت بازی و ماشین‌لرنینگ می‌رسد

۲. نحوه نوشتن جاوااسکریپت

سه راه برای نوشتن جاوااسکریپت وجود دارد:

۱. داخل تگ <script>
<script>
  console.log("Hello JS!");
</script>
۲. فایل جدا (app.js)
<script src="app.js"></script>
۳. در Node.js
node app.js

۳. متغیرها (Variables)

در جاوااسکریپت سه مدل متغیر داریم:

let - برای چیزهایی که تغییر می‌کنند
let score = 10;
score = 15;
const - برای چیزهایی که ثابت می‌مانند
const pi = 3.14;
var - قدیمی است و کمتر استفاده می‌شود
var name = "Ali";

۴. انواع داده (Data Types)

انواع داده در جاوااسکریپت به دو دسته تقسیم می‌شوند:

Primitive Types (انواع اولیه)
// number → اعداد
let age = 17;

// string → متن
let name = "Ali";

// boolean → درست/غلط
let isStudent = true;

// undefined → مقدار نداده‌ای
let x;

// null → عمداً خالی
let empty = null;

// bigint → عددهای فوق‌گنده
let bigNumber = 123456789012345678901234567890n;

// symbol → برای شناسه‌های یکتا
let id = Symbol("id");
Object (شیء)
let person = {
  name: "Ali",
  age: 17
};

۵. عملگرها (Operators)

انواع عملگرها در جاوااسکریپت:

عملگرهای ریاضی
+   -   *   /   %   **
عملگرهای مقایسه‌ای
==   ===   >   <   >=   <=
تفاوت == و ===

سه‌تایی نوع را هم چک می‌کند:

"5" == 5   // true
"5" === 5  // false

۶. دستورات شرطی

if / else
let temp = 30;

if (temp > 25) {
  console.log("گرمه!");
} else {
  console.log("خنکه!");
}
switch
let grade = "A";

switch (grade) {
  case "A":
    console.log("عالی!");
    break;
  case "B":
    console.log("خوب!");
    break;
  default:
    console.log("نا مشخص");
}

۷. حلقه‌ها (Loops)

for
for (let i = 0; i < 5; i++) {
  console.log(i);
}
while
let n = 0;
while (n < 3) {
  console.log(n);
  n++;
}
for...of (برای آرایه‌ها)
for (let x of [10, 20, 30]) {
  console.log(x);
}
for...in (برای object)
let person = {name: "Ali", age: 17};

for (let key in person) {
  console.log(key, person[key]);
}

۸. توابع (Functions)

توابع معمولی
function sum(a, b) {
  return a + b;
}
توابع فلشی (Arrow Function)
const sum = (a, b) => a + b;

۹. آرایه‌ها (Arrays)

let nums = [10, 20, 30];

nums.push(40);   // اضافه کردن
nums.pop();      // حذف آخرین مورد

۱۰. DOM چیست؟

DOM در واقع مدل درختی صفحه‌ی وب است. با جاوااسکریپت می‌توانی به عناصر صفحه دسترسی بگیری و آن‌ها را تغییر بدهی.

تغییر متن یک عنصر
document.getElementById("title").innerText = "Hello!";
فعال کردن یک دکمه
document.querySelector("button").onclick = () => {
  alert("Clicked!");
};

۱۱. مثال: یک ماشین حساب بسیار ساده

ماشین حساب ساده
نتیجه اینجا نمایش داده می‌شود
کد HTML
<input id="a" type="number" placeholder="عدد اول">
<input id="b" type="number" placeholder="عدد دوم">
<button onclick="calc()">جمع</button>
<p id="result"></p>
کد JavaScript
function calc() {
  const a = Number(document.getElementById("a").value);
  const b = Number(document.getElementById("b").value);
  document.getElementById("result").innerText = a + b;
}
